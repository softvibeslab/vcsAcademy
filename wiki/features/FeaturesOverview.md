# VCSA Features Overview

Quick reference guide to all VCSA platform features.

## 🎯 Core Features

### 1. Phase 1: Top Producer Development System

**4-Stage Progression**:
- 🌱 Stage 1: New Rep (150 pts, 1-2 weeks)
- 📈 Stage 2: Developing Rep (300 pts, 2-4 weeks)
- ⚡ Stage 3: Performing Rep (500 pts, 4-8 weeks)
- 👑 Stage 4: Top Producer (750 pts, 8-12 weeks)

**6 Training Tracks** (36 modules):
1. Pro Mindset (6 modules)
2. Discovery & Control (6 modules)
3. Value Architecture (6 modules)
4. Decision Management (6 modules)
5. Objection Mastery (6 modules)
6. Post-Decision Integrity (6 modules)

**Content Libraries**:
- 15 Deal Breakdowns (real scenario analysis)
- 20 Quick Wins (tactical scripts)
- Video lessons with Key Moves

### 2. Multi-Tenant Organization System

**Organization Features**:
- Create unlimited organizations
- Team member management
- Organization-wide analytics
- Brand customization

**School Features**:
- Create branded training academies
- Independent content libraries
- Student roster management
- School-specific analytics

### 3. Content Management

**Content Types**:
- Training courses
- Masterclasses
- Workshops
- Interviews
- Coaching modules

**Content Tools**:
- Lesson editor with video embedding
- Content upload (video, documents, images)
- AI content generation
- Video creator

### 4. Community Features

**Social Features**:
- Community feed with posts
- Likes and comments
- User following
- Event system
- Resources library

### 5. Gamification

**Engagement Features**:
- Points system (10 pts/module, 5 pts/breakdown, 3 pts/quick win)
- 11 badges with automatic awarding
- Training streaks (24h window)
- Readiness score (0-100%)
- Leaderboards (future)

### 6. AI-Powered Features

**AI Assistant**:
- Personalized recommendations
- Q&A support
- Practice scenarios
- Performance analysis

**AI Content Generation**:
- Lesson generation
- Quiz generation
- Scenario creation

## 🎨 Key Features Explained

### Readiness Score

**Calculation**:
- Video Completion (40%)
- Track Progress (30%)
- Quick Wins Applied (10%)
- Breakdowns Reviewed (10%)
- Training Streak (10%)

**Score Ranges**:
- 0-25%: Novice
- 26-50%: Developing
- 51-75%: Performing
- 76-100%: Top Producer

### Training Streak

- 24-hour window from last activity
- Resets to 0 if a day is missed
- Milestones: 7, 14, 30 days
- Badges awarded at milestones

### Watch Later / Bookmarks

**Tags**:
- `before_tour` - Review before sales tours
- `closing_help` - Closing techniques
- `objections` - Objection handling
- `discovery` - Discovery phase
- `value_build` - Value building
- `momentum` - Maintaining momentum

### Badges (11 Total)

**Completion Badges**:
- 🎯 First Steps
- 🌱 Foundation Builder
- 📈 Rising Star
- ⚡ Peak Performer
- 👑 Top Producer

**Engagement Badges**:
- 🔥 Week Warrior (7-day streak)
- ⚡ Two Week Titan (14-day streak)
- 👑 Monthly Master (30-day streak)

**Achievement Badges**:
- 📊 Deal Analyst (all breakdowns)
- 🎯 Tactician (all quick wins)
- 💯 Ready for Anything (100% readiness)

## 💡 Pro Tips

### Daily Routine

1. **Morning** (5 min): Check progress, review next assignment
2. **Before Tour** (2 min): Watch Later content (before_tour tag)
3. **After Tour** (5 min): Note what worked, find relevant module
4. **Evening** (15 min): Complete 1-2 modules

### Weekly Goals

- Complete 3-4 modules
- Review 2-3 breakdowns
- Apply 2-3 quick wins
- Engage in community

### Success Habits

- Maintain training streak
- Use Watch Before Tour feature
- Apply one technique per tour
- Share wins in community
- Celebrate badge unlocks
